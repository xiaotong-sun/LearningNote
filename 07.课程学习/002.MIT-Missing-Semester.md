## 1. Shell

| 命令               | 解释                                                         |
| ------------------ | ------------------------------------------------------------ |
| echo               | 输出字符串                                                   |
| cat                | 连接文件，并打印到标准输出设备上                             |
| cat < hello.txt    | 将hello.txt的内容传递给cat                                   |
| hello > hello.txt  | 将hello写入到hello.txt中                                     |
| hello >> hello.txt | 在hello.txt内容后面附加一个hello字符串                       |
| \|                 | 管道，将左侧的输出作为右侧的输入                             |
| tee                | 从标准输入设备读取数据，将其内容输出到标准输出设备，同时保存成文件。 |
